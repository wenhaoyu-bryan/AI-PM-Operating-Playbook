# Changelog

## 1.0.0

### Added
- Four-step AI PM Workbench with structured form-based workflow
- AI Product Brief, Workflow Spec, Evaluation Plan, Acceptance Criteria generators
- Coding Agent Handoff and CLAUDE.md Starter generators
- English and Chinese interfaces
- Local autosave with localStorage
- Three example projects (Ontology, Agent, RAG)
- JSON import and export with schema versioning
- Live document preview panel
- Mobile form/preview toggle
- Centralized field validation and completion tracking
- Illustrative example labels with credibility disclaimers

### Improved
- Unified validation and completion rules across all steps
- Export documents omit empty optional sections
- Incomplete drafts include a single clear warning
- Bilingual AI continuation prompts (Review and Implementation)
- Combined Document Pack includes project metadata header and table of contents
- Consistent Chinese terminology throughout (编码智能体)
- Product type labels are human-readable in exports

### Privacy
- User-entered project data remains in the browser (localStorage)
- No project data is uploaded to any server
