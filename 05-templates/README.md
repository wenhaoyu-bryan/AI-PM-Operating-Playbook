# Templates

Reusable frameworks and templates for AI PM work.

## Contents

- [AI PM Canvas](../AI_PM_CANVAS.md) — 12-dimension planning framework for AI product initiatives (at repo root)
- [AI PM Case Study Template](AI_PM_CASE_STUDY_TEMPLATE.md) — Structured template for documenting AI PM projects
- [AI PRD Lite Template](AI_PRD_LITE_TEMPLATE.md) — Lightweight PRD template for PM-led AI prototypes

## When to use this

- When starting a new AI product initiative — use the Canvas
- When documenting a completed project — use the Case Study Template
- When scoping a prototype — use the PRD Lite Template
